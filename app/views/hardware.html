Wboard Pro

Wboard Pro is a Wifi development board based on Arduino MEGA. With a built-in ESP8266 chip (wifi firmware burnt), you can drive ESP8266 to connect wifi by controlling the serial ports and visit the internet. WBoard Pro supports IoTgo Arduino Library to do a rapid development.

Hardware Library Support

IBoard / IBoard Pro

IBoard and IBoard Pro are two Ethernet development boards based on Arduino and Arduino MEGA. With a W5100 ethernet chip, they support twisted-pair cable to connect router and visit the internet. You can make a simple development to communicate with IoT by utilizing Arduino Ethernet library. Among them, IBoard Pro supports IoTgo Arduino Library to do a rapid development.

Hardware Library Support

Gboard / Gboard Pro

Gboard and Gboard Pro are two GSM/GPRS development boards based on Arduino and Arduino MEGA. With an on-board SIM900 module, you can plug an SIM card and connect to IoTgo server through GPRS network. You also can make a simple development to communicate with IoTgo by utilizing Arduino GSM library. Among them, GBoard Pro supports IoTgo Arduino Library to do a rapid development.

AllWinner A20 Series

The boards on the left using Allwinner A20 chip support IoTgo Segnix Library to develop smart device or smart gateway. Among them, Core EVB is 100% compatible with Segnix I/O configurable files, while other boards need to modify the configure files and these modifications won't have an effect on the communication codes of Ethernet and IoTgo.

Raspberry Pi

Raspberry Pi is a very popular credit card size computer. You can use it as an IoTgo smart device or smart gateway. The Segnix we provide supports Raspberry Pi. You also can use our IoTgo Segnix Library in Raspberry Pi to do a rapid development.

BB Board

BB Board is a credit card size computer with TI XXX chip. You can use it as an IoTgo smart device or smart gateway. The Segnix development for BB Board is under way, and soon we will finish IoTgo Segnix Library that supports BB Board to do a rapid development.

WRTNode

WRTNode is a popular OpenWrt development board. You can use it as an IoTgo smart device or smart gateway. The Segnix development for WRTNode is under way. If you are interested in helping us to make a progress in this part, please contact us.

Intel Galileo

Intel Galileo is an X86 single board computer compatible with Arduino. You can use it as an IoTgo smart device or smart gateway. If you develop Galileo like the way you develop arduino, our IoTgo Arduino Library is suitable for your development. The Segnix version for Galileo is not in our plan, but we welcome the developers who are interested in this project to improve the code.
